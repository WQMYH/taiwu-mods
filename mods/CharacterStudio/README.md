# CharacterStudio

统一人物创建与定制 MOD。

## 已实现

- F9 游戏内面板批量创建太吾村民。
- 使用原版 `RecruitCharacterData` 招募管线。
- 性别、年龄、魅力、姓名定制。
- 创建后姓名与身体类型定制。
- 六维、技艺资质、武学资质的最低值/覆盖值模式。
- 特性显式白名单添加和移除。
- 健康、道德、服装、关系、好感配置。
- 太吾创建点数解除。
- 太吾、谷中密友、手动创建村民分别设置开关。
- 谷中密友数量可配置，并带递归保护。
- 可选处理原版招募人物及太吾村初始核心人物。
- 创建前自动扩展村庄人口容量。

## 安全边界

- 不使用“特性 ID 取模”判断正面特性。
- 不全局 Patch 住宅分配。
- 不替换原版村庄家庭生成。
- 双性恋字段依赖私有字段，默认关闭。
- 初始资源、地图资源和物品奖励不属于人物创建核心，暂未混入创建事务。
