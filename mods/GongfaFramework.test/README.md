# GongfaFramework.test

版本 `0.1.0.0`，适配《太吾绘卷》`1.0.40`。

按 `Ctrl+F8` 打开游戏内功法框架。M1 可读取、搜索、筛选及导出全部功法，并查看关联秘籍、正逆练特效和原始 JSON；简易编辑器生成字段补丁，正式补丁重启后生效。

用户文件始终位于本 MOD 的 `UserData` 中。导入文件放入 `UserData/imports`，导出位于 `UserData/exports/<时间>`，备份位于 `UserData/backups/<时间>`，日志位于 `UserData/logs`。
