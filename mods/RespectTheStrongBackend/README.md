# 敬重强者·后端修复版

针对当前太吾绘卷多 DLL 后端架构重写的 RespectTheStrongBackend。

各功能会在启动时独立检查目标方法。游戏更新导致某个目标失效时，仅禁用对应功能并写入 `GameData.log`。

构建运行 `build.bat`，部署运行 `deploy.bat`。可通过 `TAIWU_GAME_DIR` 指定游戏目录。

## 日志

模组会同时写入游戏后端总日志和独立日志：

`<游戏目录>/Logs/RespectTheStrongBackend.log`

独立日志记录每次会话、设置快照、补丁安装状态、关键功能执行结果和完整异常堆栈。文件达到
10 MB 后会在下次启动时按时间归档。开启“输出诊断日志”后，还会记录人物生成、倍率计算、
晋升修正及跳过路径等高频明细。
