# 太吾村建筑蓝图工具 (CopyBuildingModernized.Another)

## 📖 功能介绍

本MOD为太吾绘卷提供太吾村建筑蓝图的导入、导出和宽度转换功能，支持扩展村庄规模。

### 核心功能

1. **导入蓝图** - 从外部.bin文件导入建筑布局
2. **导出蓝图** - 将当前太吾村建筑布局保存为.bin文件
3. **蓝图转换** - 转换蓝图到不同的村庄宽度（18-126）
4. **自动配置** - 可选的产业领袖分配和技艺补强

---

## 🎮 使用方法

### 访问界面

1. 进入游戏中的**产业管理**界面
2. 找到 **"太吾村建筑管理"** 按钮（位于原有按钮区域）
3. 点击按钮展开/收起子菜单

### 子菜单功能

展开后可见4个功能按钮：

- **导入蓝图** - 选择.bin文件导入建筑布局
- **导出蓝图** - 保存当前建筑布局为.bin文件
- **蓝图转换** - 转换现有蓝图到目标宽度
- **目标大小** - 输入框，设置转换的目标宽度（默认24）

### 配置选项

在MOD管理界面中可调整以下配置：

| 配置项                 | 说明                                        | 默认值  |
| ---------------------- | ------------------------------------------- | ------- |
| 导入时自动安排产业领袖 | 为需要领袖的建筑分配合适村民                | ✅ 开启 |
| 产业领袖技艺补强       | 补充领袖对应技艺等级（-1关闭，0-8表示等级） | 8       |
| 导入时清理操作状态     | 清空操作进度、工匠订单等运行时状态          | ✅ 开启 |

## ⚠️ 注意事项

### 导入后崩溃问题

如果导入蓝图后游戏崩溃，请检查：

1. **配置项是否开启** - 确保"导入时清理操作状态"已启用
2. **蓝图兼容性** - 确认蓝图文件大小与目标村庄匹配
3. **存读档刷新** - 导入后如建筑未显示，尝试存读档一次

### 转换失败问题

常见原因：

- 原始.bin文件损坏或不完整
- 目标宽度超出范围（必须在18-126之间）
- 2x2建筑的根格索引错误

解决方法：

1. 重新导出原始蓝图
2. 检查目标宽度是否合法
3. 查看日志确认具体错误

## 💡 使用建议

## ❓ 常见问题

**Q: 如何恢复默认配置？**
A: 在MOD配置界面点击"重置为默认值"。

**Q: 转换后的蓝图能反复使用吗？**
A: 可以，但仅适用于对应宽度的太吾村。

**Q: 为什么导入后建筑不显示？**
A: 尝试存读档一次刷新游戏状态。
